# Heart-Disease-UCI

This exploratory analysis was carried out as part of an academic group project.
The interest of this project was to become familiar with machine learning tools on python, treating real unprocessed datas.
Subject : is it possible to predict the onset of heart disease based on some given informations ?

You will be able to find the data used in the files that are in a .data format.
